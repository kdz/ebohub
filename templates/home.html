About EboHub

EboHub is an SMS-based service for an Ebola-striken area with a population using primarily feature-phones, not smart-phones. This web site itself, exposed here via Admin pages, is mostly meant for administrators rather than general people using smart phones or desktop browsers, although it could be quite easily re-purposed for such usage.

Text #help to 512-942-0915 (assuming the Twilio account has money left in it!)
  • Sending "i'm sick" registers anyone as a Patient
  • You can also use the Admin pages to register yourself + phone as Health Care Worker
  • If you have more than one phone number, register one as Patient ("i'm sick") and the other as HC Worker (via Admin Pages / HC Worker / Create)
    • Register both phones within the same "Chiefdom" (i.e. geographic area)
    • Try the different SMS options from each phone and watch the SMS responses and web updates
